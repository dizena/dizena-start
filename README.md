### web-mgr
是基于VUE2的管理端UI
####安装方法？

进入模板目录
<br>
cd web-mgr

安装项目依赖，安装失败可用 cnpm 或 yarn
<br>
npm install

开启服务器
<br>
npm run serve
npm run dev
npm run test
npm run prod

执行构建命令，生成的dist文件夹放在服务器下即可访问
<br>
npm run build
